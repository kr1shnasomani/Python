You are given an integer array 'arr' of size 'N'.

**Note:**

Change in the input array itself. You don't need to return or print the elements.

# Detailed explanation ( Input/output format, Notes, Images )

## Constraints :

- 0 <= N <= 10^3
- 0 <= arr[i] <= 10^5

Time Limit: 1 sec

---

```text
5
9 3 6 2 0
```

## Sample Output 1:

```text
0 2 3 6 9
```

## Sample Input 2:

```text
4
4 3 2 1
```

## Sample Output 2:
